# 智慧医疗 - 药品出入库系统 用户使用手册

## 1. 系统概述
本系统是一套集成了医生工作站、药师工作站及后台管理功能的综合性药品库存管理系统。旨在规范药品流通环节，实现从处方开具、药品发放到库存盘点的全流程数字化管理。

## 2. 登录系统
*   **启动软件**：双击运行程序（`.exe`）。
*   **登录界面**：输入分配的用户名和密码。
*   **角色识别**：系统会自动根据账号角色（管理员、医师、药师）进入相应的工作台。
*   **注意事项**：初始账号请联系管理员获取。

---

## 3. 管理员端 (Administrator)

管理员主要负责系统基础数据的维护和系统运行状态的监控。

### 3.1 医生/用户管理
*   **功能入口**：侧边栏 -> [医生管理]
*   **查看用户**：列表显示所有系统用户（ID、用户名、姓名、角色、状态）。
*   **新增用户**：
    1. 点击顶部“新增用户”按钮。
    2. 填写用户名、真实姓名。
    3. 选择角色（DOCTOR-医师, PHARMACIST-药师, ADMIN-管理员）。
    4. 设置初始密码。
    5. 点击“保存”。
*   **编辑用户**：点击列表右侧“编辑”按钮，可修改姓名、角色或重置密码（留空则不修改）。
*   **启/停用**：点击“启用/停用”按钮，可快速控制账号登录权限。

### 3.2 操作日志
*   **功能入口**：侧边栏 -> [操作日志]
*   **查询日志**：在顶部搜索框输入关键字，点击“查询”筛选日志。
*   **查看详情**：点击列表右侧“查看”按钮，查看详细的操作数据（JSON格式化显示）。
*   **清空日志**：点击“清空所有日志”按钮（慎用），将删除所有历史操作记录，此操作不可恢复。

---

## 4. 医师端 (Doctor)

医师主要负责患者接诊、处方开具及查询药品信息。

### 4.1 就诊登记 (开处方)
*   **功能入口**：侧边栏 -> [就诊登记]
*   **患者信息**：填写姓名、性别、年龄、就诊日期。
*   **临床诊断**：
    *   **选择诊断**：点击输入框右侧“▼”或直接输入关键字搜索，从下拉列表中选择。
    *   **自定义诊断**：如列表中没有，选择列表第一项“--- 自定义诊断 ---”，输入自定义内容。
    *   **新增诊断**：点击“+”按钮可快捷添加常用诊断类型到系统。
    *   **删除诊断**：点击“-”按钮可管理删除不用的诊断类型。
*   **处方药品**：
    *   **搜索药品**：在药品框输入名称/拼音，系统实时筛选，支持下拉选择。
    *   **添加药品**：输入数量后点击“添加药品”加入下方列表。
    *   **删除药品**：在下方已选列表中点击“删除”移除误加药品。
*   **提交**：确认信息无误后，点击“提交就诊记录”，状态变为“待发药”。

### 4.2 就诊记录
*   **功能入口**：侧边栏 -> [就诊记录]
*   **查询记录**：支持按患者姓名、日期范围筛选历史记录。
*   **查看状态**：
    *   `待发药`：已提交，等待药师处理。
    *   `已完成`：药师已发药。
    *   `已退回`：药师因库存不足或其他原因退回。
*   **修改重提**：对于状态为“已退回”的记录，点击“修改重提”按钮，可修改处方内容后再次提交。

### 4.3 药品查询
*   **功能入口**：侧边栏 -> [药品查询]
*   **实时库存**：查询所有药品的实时库存、单价及规格信息，辅助开药决策。

### 4.4 统计报表
*   **功能入口**：侧边栏 -> [统计报表]
*   **查看报表**：查看药品进销存报表和运营统计报表（权限受限，通常只读）。

## 5. 药师端 (Pharmacist)

药师负责药品发放、采购入库、库存盘点及药品基础信息管理。

### 5.1 待发药处理
*   **功能入口**：侧边栏 -> [待发药]
*   **处理处方**：列表显示医师提交的处方。
*   **发药**：核对库存充足后，点击“发药”，系统自动扣减库存。
*   **退回**：如库存不足或处方有误，点击“退回”并填写原因，记录将返回给医师。

### 5.2 库存盘点
*   **功能入口**：侧边栏 -> [库存盘点]
*   **流程**：
    1.  **生成任务**：选择月份，点击“生成/获取盘点任务”，系统锁定当前系统库存快照。
    2.  **实盘录入**：
        *   在表格中“实盘数量”和“备注”栏，双击“点击填写”进行录入。
        *   双击时会自动清除占位符，录入数字后回车保存。
        *   系统自动计算“差异”（实盘 - 系统库存），差异不为0时显示红色。
    3.  **完成盘点**：所有数据录入完毕后，点击“完成本次盘点”，任务归档。
*   **修正**：如需修改已生成的任务，可点击“重新修改”（需任务未完成状态）或“删除任务”重置。

### 5.3 药品管理
*   **功能入口**：侧边栏 -> [药品管理]
*   **基础信息**：新增、修改、删除药品信息（名称、规格、单位、单价）。
*   **库存修正**：支持直接修改库存数量（会弹出警告，建议通过“药品购进”增加库存）。

### 5.4 药品购进
*   **功能入口**：侧边栏 -> [药品购进]
*   **购进录入**：
    1.  选择药品（自动填充单位）。
    2.  输入数量、总金额（自动计算单价）。
    3.  点击“添加待提交”加入暂存列表（粉色背景）。
*   **批量提交**：确认所有购进项无误后，点击“提交新增记录”，库存将正式增加。
*   **查看历史**：下方列表绿色背景项为历史已入库记录。

### 5.5 统计报表
*   **功能入口**：侧边栏 -> [统计报表]
*   **药品进销存月报**：统计本月期初、购进、使用、期末的理论与实盘数据。
*   **运营统计月报**：统计每日就诊人数、药品费用、人均费用等。
*   **导出**：支持将报表导出为 Excel 文件 (`.xlsx`)。
